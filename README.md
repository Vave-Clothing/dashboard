# Vave Dashboard

[![CI](https://github.com/Vave-Clothing/dashboard/actions/workflows/integrate.yml/badge.svg?branch=main&event=push)](https://github.com/Vave-Clothing/dashboard/actions/workflows/integrate.yml)

This is the admin dashboard from [Vave Clothing](https://vave-clothing.de/), made with NextJS, Typescript and Matine.

## Getting started

1. Clone Repo
2. Run `yarn`
3. Run `yarn dev` to start the dev server
4. Colaborate
